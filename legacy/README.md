# Legacy Conecta Cuatro

Original static version of the game, preserved for reference and comparison.

## Contents

- `conecta4.html` — main game page
- `instruccion.html` — instructions page
- `css/` — original stylesheets
- `js/conecta4.js` — original game logic

## How to open locally

Open `conecta4.html` in a browser, or serve this folder with any static file server.

> **Note:** Image assets (`imagenes/`) were not included in the repository. Buttons and board still work; finger/decoration images may be missing.

The active app lives in the project root (Next.js). Use `pnpm dev` from the repository root.
