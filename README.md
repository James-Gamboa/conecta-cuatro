# Conecta Cuatro (Next.js)

Modernized Connect Four built with Next.js 16.2, TypeScript, Tailwind CSS, shadcn/ui, GSAP, and Lenis.

## Requirements

- Node.js 20+
- pnpm 9+

## Scripts

```bash
pnpm dev          # development server
pnpm build        # production build
pnpm start        # production server
pnpm lint         # ESLint
pnpm type-check   # TypeScript check
```

## Legacy version

The original HTML/CSS/JS app is archived in [`legacy/`](./legacy/) for reference.

## Deploy on Vercel

Use the repository root and pnpm as the package manager.
